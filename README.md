# Debian Installer Companion

Debian installation companion with hardware compatibility checks.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install debian-installer-companion
```
